Alur cepat

  1. Klik Start & Open Window di atas — jendela Label Studio muncul.
  2. Pertama kali: klik Sign Up, buat akun (email + password apa saja, offline).
  3. Create Project → beri nama (misal "Socket Holder 1160").
  4. Tab Data Import → Upload gambar dari folder dataset (path di atas).
  5. Tab Labeling Setup → template Object Detection with Bounding Boxes.
  6. Tambahkan label kelas: OK, waterspot, scratch, rust, dst.
  7. Save → klik Label All Tasks → mulai anotasi.
  8. Selesai: Export → format YOLO with images. Extract, copy folder labels/ ke dataset/labels/train/.
