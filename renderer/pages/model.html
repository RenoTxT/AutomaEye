Annotation (Label Studio embedded)

Klik tombol di bawah untuk buka Label Studio langsung di dalam aplikasi (embedded). Server jalan di background.

Pertama kali: buat akun (offline, disimpan lokal) → New Project → Data Import upload gambar dari folder dataset → Labeling Setup pilih Object Detection with Bounding Boxes → set kelas (OK, waterspot, dll) → mulai anotasi.

Split Dataset (Train / Val / Test)

Langkah 2: setelah semua gambar dianotasi, bagi gambar asli (beserta anotasinya) jadi train / val / test. Geser 2 titik slider untuk atur proporsi. Baru setelah ini augmentasi dijalankan — dan hanya untuk train.

Train: 70% Val: 20% Test: 10%

Clean & Rebuild: hapus gambar augmentasi lama + betulkan nama label + buang gambar tanpa label (mis. yang di-skip saat anotasi), lalu split. Pakai kalau training bilang "no labels" atau ada gambar kosong di dataset.

Data Augmentation

Generate variasi gambar per opsi (tidak dicampur). Contoh: centang Rotate + Blur, multiplier=2, dari 1 gambar → 4 output baru (2 rotate, 2 blur), gambar asli tetap ada. Label ikut ditransform otomatis. Hanya gambar di train yang diaugmentasi.
